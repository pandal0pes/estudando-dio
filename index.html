Tag title

[Voltar ao topo]

A tag title é utilizada inicialmente dentro da head do site, para adicionar/definir um titulo (como diz o nome da tag) na aba do site. Também é utilizada como atributo dentro de uma tag, por exemplo, a tag h1 para que apareça uma caixa de texto apresentando informações ao usuario ao passar o mouse em cima do texto/link.
com base no meu pequeno conhecimento de inicio de curso.

“O elemento HTML title (Elemento HTML Título) define o título do documento, mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas. ”
Fonte: https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/title

Tag h1

[Voltar ao topo]

A tag h1 é usada para escrever um cabeçalho com formatação(Tamanho, Espaçamento, etc..) para diferenciar o título principal da página dos demais conteúdos.

“H1 é a abreviação de “Heading Tag 1” e também é a primeira e mais importante das seis tags utilizadas para definir e diferenciar os títulos do código HTML de uma página web. Para que serve a tag H1? A principal função da tag H1 é diferenciar o título principal do restante do conteúdo. ”
Fonte: https://i.workana.com/guias/o-que-e-tag-h1/
